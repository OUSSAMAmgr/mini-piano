# mini-piano

Mini Piano est un projet web interactif qui transforme votre clavier en batterie virtuelle.  
Chaque touche correspond à un son unique : pas, chant d’oiseau, coup de feu, électricité, moteur, explosion, vent, cloche, clic de porte.

## Fonctionnalités
- Jouer des sons en appuyant sur les touches du clavier
- Animation visuelle des touches
- Design responsive pour PC, tablette et mobile
- Déploiement sur GitHub Pages

## Comment utiliser
1. Ouvrir le site dans votre navigateur
2. Appuyer sur les touches A à I pour jouer des sons
3. Amusez-vous à créer vos propres rythmes !

## Technologies utilisées
- HTML, CSS, JavaScript
